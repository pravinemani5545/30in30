Five dimensions (0-20 each):
- Clarity: Can a stranger understand what this does in 5 seconds?
- Specificity: Does it name the actual customer and actual problem?
- Differentiation: Does it imply why this is different or better?
- Audience fit: Is the language right for investors/customers/press?
- Memorability: Will they remember it in 24 hours?

Calibration:
- Average first-draft score: ~23/100
- "We make software for businesses" = ~8/100
- "Stripe for X" pattern: clarity 18, specificity 12, differentiation 6
- YC top tier: typically 78-88/100 after 3 rewrites

Common failure modes:
- Too broad: no named customer, no named problem
- Jargon-heavy: loses non-technical investors
- Feature description instead of outcome description
- Missing the "why now" or "why us" signal
- Forgettable verbs: "enables", "allows", "helps", "leverages"
